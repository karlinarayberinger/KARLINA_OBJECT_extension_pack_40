SOUND_TRACK_LOOPER_THREE

Play one (or multiple) sound pattern(s) (whose duration(s) is(are) finite) for some indefinite number of cycles.

(If you are experiencing this app via Tor on a mobile device, please ensure that audio is enabled in that browser's settings).

(Note that there may be a delay of several seconds before a sound file is played due to network latency across the Tor network).

 01_FEBRUARY_2025: This application uses the same essential logic as what is featured in SOUND_TRACK_LOOP_COUNTER_TWO (and which is also hosted on the same .onion server as what hosts this web page). The only difference between SOUND_TRACK_LOOP_COUNTER_TWO and SOUND_TRACK_LOOP_COUNTER_THREE is that SOUND_TRACK_LOOP_COUNTER_THREE features different MP3 files than what SOUND_TRACK_LOOP_COUNTER_TWO features and SOUND_TRACK_LOOP_COUNTER_THREE has this additional static text on the application web page interface. Additionally, SOUND_TRACK_LOOP_COUNTER_THREE is not featured in SOFTWARE_ENGINEERING_PORTFOLIO whereas SOUND_TRACK_LOOP_COUNTER_TWO is due to the fact that SOUND_TRACK_LOOP_COUNTER_THREE features a larger and messier (i.e. not entirely karbytes-originated) collection of sound files than what SOUND_TRACK_LOOP_COUNTER_TWO features (but karbytes thinks that the MP3 file collection which can be experienced through SOUND_TRACK_LOOP_COUNTER_THREE makes for a decent background noise generator to help manage misophonia (i.e. strong dislike of specific auditory stimuli occurring in one's environment and beyond one's control)). 

1) Select one or multiple sound files from the list below.

2) Click the start_sound_track_looper() button to start playing the selected sound file(s).
