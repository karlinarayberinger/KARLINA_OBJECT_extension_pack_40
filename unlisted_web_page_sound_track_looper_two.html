SOUND_TRACK_LOOPER_TWO

Play one (or multiple) sound pattern(s) (whose duration(s) is(are) finite) for some indefinite number of cycles.

(If you are experiencing this app via Tor on a mobile device, please ensure that audio is enabled in that browser's settings).

(Note that there may be a delay of several seconds before a sound file is played due to network latency across the Tor network).

1) Select one or multiple sound files from the list below.

2) Click the start_sound_track_looper() button to start playing the selected sound file(s).

This section will be updated with information about selected sound tracks after the start button is clicked.
